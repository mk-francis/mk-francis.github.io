What the app does

The web app is designed for creator teams that need a clean publishing workflow instead of switching between multiple TikTok accounts and manual upload steps.

  • Sign in with TikTok and authorize only the required scopes.
  • Upload local video files before publishing.
  • Publish directly to the authorized TikTok profile.

Publishing workflow

The current integration uses TikTok Login Kit and Content Posting API Direct Post. The website URL and callback URL below match the configuration submitted in TikTok Developer Portal.

Step 1

The user starts sign-in from the MOKOM web app and authorizes TikTok access.

Step 2

TikTok redirects the user to the verified callback URL on this domain.

Step 3

The user selects a local video file for upload through the publishing flow.

Step 4

The app submits the upload and publish actions to TikTok for direct posting.
